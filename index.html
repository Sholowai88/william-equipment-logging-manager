📘 User Guide

  • Real-time Filtering — Type in any search box, equipment list updates instantly.
  • Select / Compare — Check equipment cards → click "Compare" to view side-by-side details and logs.
  • Add/Edit Equipment — Click New Equipment or edit icon ✏️. Fill form; optionally add a log.
  • Edit/Delete Logs — Inside Logs window, click "Edit" or "Delete" on any log row.
  • Backup & Restore — Backup saves JSON. Restore gives option to Overwrite or Merge.
  • Export (CSV, Excel, PDF, JPEG) — Export selected equipment logs.
  • Audit Log — Every action is tracked with timestamp. Click Audit Log button to view all changes.
  • Cloud Sync — Click "Sync to Cloud" to save all data to Supabase. "Load from Cloud" retrieves the latest cloud data.
  • Save & Refresh — Always save changes. Refresh reloads from storage.
  • 📊 Current Database: 44 Equipment Units, 247 Maintenance Records

All user actions are automatically recorded in the audit trail.
